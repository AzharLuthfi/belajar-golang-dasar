# golang-dasar

Repository ini berisi catatan, latihan, dan contoh kode untuk mempelajari dasar-dasar bahasa Go (Golang).

## Tujuan
- Memahami sintaks dasar Go
- Mempelajari konsep fundamental seperti variabel, kontrol alur, function, dan struct
- Membiasakan diri dengan cara kerja Go secara bertahap

## Isi Repository
Materi dalam repository ini akan berkembang seiring proses belajar, mulai dari:
- Dasar bahasa Go
- Contoh program sederhana
- Eksperimen kecil untuk memahami konsep Go
